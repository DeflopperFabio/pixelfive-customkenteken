# pixelfive-customkenteken

pixelfive-customkenteken is een script voor FiveM waarmee voertuigen **custom kentekens** krijgen.  
Het script vervangt runtime textures met jouw eigen plaatjes (.png) en werkt volledig client-side.  

## Features

- Ondersteunt meerdere custom kentekens  
- Vervangt runtime textures met eigen afbeeldingen  
- Klaar voor NL/RP servers  
- Flexibel en uitbreidbaar

## Installatie

1. Plaats de map `pixelfive-customkenteken` in je `resources` folder  
2. Voeg toe aan `server.cfg`:
   ```text
   ensure pixelfive-customkenteken
